# –ó–∞–¥–∞—á–∞ 2

–Ø–∑—ã–∫ –ø–∞–ª–∏–Ω–¥—Ä–æ–º–æ–≤, –∫–æ—Ç–æ—Ä—ã–µ –∑–∞–æ–¥–Ω–æ –Ω–µ —è–≤–ª—è—é—Ç—Å—è —Å–ª–æ–≤–∞–º–∏ –≤–∏–¥–∞ ww.

–°–ª–æ–≤–æ $a^{n!+n} b^n a^{2n} b^n a^{n!+n}$

–ù–∞–∫–∞—á–∫–∞ –±–ª–æ–∫–æ–≤ b, ab, ba –∏ –∫—Ä–∞–π–Ω–∏—Ö a –≤—ã–≤–µ–¥–µ—Ç —Å–ª–æ–≤–æ –∏–∑ –ø–∞–ª–∏–Ω–¥—Ä–æ–º–æ–≤. –ù–∞–∫–∞—á–∏–≤–∞–µ–º —Å–µ—Ä–∏–¥–∏–Ω—É.

$a^{n!+n} b^n a^{2n -i + ik} b^n a^{n!+n}$

–Ω—É–∂–Ω–æ –Ω–∞–π—Ç–∏ —Ç–∞–∫–æ–µ k, —á—Ç–æ —Å–ª–æ–≤–æ —Å—Ç–∞–Ω–µ—Ç –∏–º–µ—Ç—å –≤–∏–¥ ww:

$2(n! + n) = 2n - i + ik$

$2n! = - i + ik$

$2n! = i(k - 1)$

$k = \frac{2n!}{i} + 1$

–¥–µ–ª–µ–Ω–∏–µ —Ñ–∞–∫—Ç–æ—Ä–∏–∞–ª–∞ –Ω–∞ i –≥–∞—Ä–∞–Ω—Ç–∏—Ä–æ–≤–∞–Ω–Ω–æ –±—É–¥–µ—Ç —Ü–µ–ª—ã–º (—ç—Ç–æ –∂–µ —Ñ–∞–∫—Ç–æ—Ä–∏–∞–ª). –Ω–∞—à–ª–∏ k.
